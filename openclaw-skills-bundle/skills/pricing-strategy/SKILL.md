# 定价策略

## 常用定价法

### 1. 成本加成
```
价格 = 成本 × (1 + 利润率)
```

### 2. 价值定价
根据用户感知价值定价

### 3. 竞争定价
参考竞争对手定价

### 4. 渗透定价
低价进入市场

### 5. 撇脂定价
高价高利润

## 定价技巧

### 价格锚点
- 设置高价选项作为锚点
- 中间价位更易被选择

### 数字游戏
- 99 / 49 / 9 
- 1999 → 2000

### 套餐定价
```
基础版：99
专业版：199（最推荐）
企业版：499
```

### 分期付款
降低用户付费痛感

## 数字化产品定价

### SaaS定价
- 按用户数收费
- 按功能分层
- 年付享折扣

### 常见策略
- 免费试用
- 限时优惠
- 首次折扣
- 续费优惠
