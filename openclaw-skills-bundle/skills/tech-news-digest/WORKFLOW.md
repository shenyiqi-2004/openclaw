# 科技新闻简报 - 工作流程

> 核心原则：我只做裁判，不做运动员

---

## 质量标准（每篇新闻）

| 部分 | 字数要求 |
|------|----------|
| 核心事件 | 300+ 字 |
| 深入分析 | 300+ 字 |
| 我的观点 | 300+ 字 |
| **每篇总计** | **900+ 字** |

---

## 工作流程

```
1. Cron 9:00 AM
   ↓
2. 自动抓取新闻 → JSON
   ↓
3. 子 agent 筛选 + 写初稿
   ↓
4. 提交给我审核
   ↓
5. 我审核（裁判）
   ↓
   ├─ ❌ 不通过 → 退回子 agent 修正
   └─ ✅ 通过 → 我发送
```

---

## 审核标准

```
□ 1. 都是 AI 相关？
□ 2. 核心事件 300+ 字？
□ 3. 深入分析 300+ 字？
□ 4. 我的观点 300+ 字？
□ 5. 没有水文？
□ 6. 格式统一？
```

---

## 禁止事项

- ❌ 自己审核自己的作品
- ❌ 用"昨日已分析"偷懒
- ❌ 用"由于篇幅有限"偷懒
- ❌ 复制粘贴旧内容
