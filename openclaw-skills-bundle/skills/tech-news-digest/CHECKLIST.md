# 科技新闻简报 - 质量检查清单

> 核心原则：我只做裁判，不做运动员

---

## 角色分工

| 角色 | 工作 |
|------|------|
| **我（裁判）** | 审核质量、决定是否发送 |
| **子 agent（运动员）** | 筛选新闻、读取原文、写初稿 |

---

## 审核标准（裁判使用）

我作为裁判，必须检查子 agent 提交的分析：

```
□ 1. 都是 AI 相关？
□ 2. 核心事件 300+ 字？
□ 3. 深入分析 300+ 字？
□ 4. 我的观点 300+ 字？
□ 5. 没有"昨日已分析"（必须是新写的）？
□ 6. 没有水文？
□ 7. 格式统一？
□ 8. 没有偷懒句式？
```

❌ 任何一项不通过 → 退回子 agent 修正

**每篇总计：900+ 字**

**字数检查方法**：
- 核心事件：~300 字（说清楚发生了什么）
- 深入分析：~300 字（为什么会发生、背景、影响）
- 我的观点：~300 字（独立思考、独特见解）

---

## 筛选标准

### 主题筛选（必须满足）

| 检查项 | 说明 |
|--------|------|
| ✅ AI 相关 | LLM、AI Agent、AI编程、AI行业、AI安全 |
| ✅ 科技行业重大新闻 | 行业趋势、公司动态 |
| ❌ 排除 | 纯围棋、纯生物科学、纯政治新闻 |

### 来源筛选

| 检查项 | 说明 |
|--------|------|
| ✅ 优先 | 官方博客（OpenAI、Google）、知名科技媒体（TechCrunch、The Verge） |
| ⚠️ 次选 | 个人博客（Simon Willison、Gary Marcus） |
| ❌ 排除 | 来源不明的聚合站 |

---

## 内容标准

### 每篇必须满足（全部打勾）

- [ ] **读取原文** - 不是只看标题，是真的打开链接读取内容
- [ ] **500+ 字** - 深度分析至少 500 字
- [ ] **核心事件** - 用自己的话概括新闻在讲什么
- [ ] **深入分析** - 为什么会发生？意味着什么？
- [ ] **独立观点** - 有自己的思考，不是复制粘贴

### 禁止出现

- [ ] ❌ 车轱辘话（来回说同样的意思）
- [ ] ❌ 凑字数（硬凑长度）
- [ ] ❌ 复制原文（用自己的话写）
- [ ] ❌ 没有观点（只是总结，没有看法）

---

## 格式标准

### 统一格式

```
### 标题 ⭐⭐⭐
链接

📰 核心事件：
（1-2 句话概括）

🔍 深入分析：
（3-5 段分析）

💡 我的观点：
（1-2 段独立思考）
```

### 评分标准

| 等级 | 字数 | 深度 | 观点 |
|------|------|------|------|
| ⭐⭐⭐ | 800+ | 非常深入 | 独特见解 |
| ⭐⭐ | 500-800 | 比较深入 | 有自己的想法 |
| ⭐ | 300-500 | 简单总结 | 较少的观点 |

## 发送前自检

### 快速检查

```
□ 今天有 10 篇新闻？
□ 都是 AI 相关？
□ 没有混入围棋/生物/政治？
□ 每篇都有 500+ 字？
□ 每篇都有独立观点？
□ 格式统一？
□ 错别字检查？
□ 没有"简述"或"由于篇幅有限"？
```

### 禁止出现的句式

❌ 禁止：
- "由于篇幅有限，简述如下"
- "以下简单总结"
- "剩余新闻简述"
- "篇幅原因，只列出标题"

✅ 正确做法：
- 明确说"只找到 X 篇符合标准的新闻"
- 或者"以下 Y 篇信息不足，无法深入分析"

### 如果发现问题

```
❌ 某篇不是 AI 相关 → 替换
❌ 字数不够 → 补充分析
❌ 没有观点 → 添加观点
❌ 格式不统一 → 统一格式
❌ 不够 10 篇 → 说明"只找到 X 篇"
```

## 审核流程

### 子 agent 职责
1. 筛选新闻
2. 读取原文
3. 写初稿
4. 提交审核

### 人工审核职责
1. 检查主题相关性
2. 检查内容深度
3. 检查观点独立性
4. 决定是否发送

### 修正流程
```
审核不通过 → 说明问题 → 子 agent 修正 → 再次审核
```

## 示例对比

### ❌ 不合格的分析

> 这篇新闻讲的是 OpenAI 获得了 110 亿美元投资，估值达到 7300 亿美元。这是一个很大的数字，说明 OpenAI 很有钱。我觉得这对 AI 行业有影响。

问题：
- 只复制了标题
- 没有深入分析
- 观点太浅

### ✅ 合格的分析

> OpenAI 宣布获得 110 亿美元投资，估值 7300 亿美元。这是 AI 行业历史最大融资。
> 
> **深入分析**：...
> 
> **我的观点**：...
