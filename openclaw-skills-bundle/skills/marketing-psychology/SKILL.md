# 营销心理学

## 核心原理

### 1. 社会认同
- 展示用户评价
- 销量数据
- "XX人已购买"

### 2. 稀缺性
- 限时优惠
- 限量供应
- 库存紧张

### 3. 权威
- 专家背书
- 资质证书
- 合作案例

### 4. 承诺与一致
- 免费试用
- 小额首单
- 会员积分

### 5. 喜好
- 明星代言
- 美丽设计
- 共同兴趣

### 6. 互惠
- 免费赠品
- 返现
- VIP服务

## 应用场景

### 文案钩子
```
[社会认同]：已有1000+企业选择
[稀缺性]：仅剩23个名额
[权威]：XX行业认证
[互惠]：首单立减50元
```

### 定价策略
- 锚定效应：设置高价锚点
- 厌恶损失：强调错过优惠
- 虚拟价值：赠品价值

## 检查清单

- [ ] 是否有社会认同证据
- [ ] 是否制造紧迫感
- [ ] 是否有权威背书
- [ ] 是否有互惠诱饵
