# 网页设计指南

## 设计原则

### 1. 布局
- 栅格系统：12列栅格
- 常用宽度：1440px / 1200px / 750px
- 间距：4px / 8px / 16px / 24px / 32px / 48px / 64px
- 响应式断点：576px / 768px / 992px / 1200px

### 2. 色彩
- 主色：品牌主色（建议#000000 - #FFFFFF）
- 辅助色：灰度色系
- 强调色：用于CTA和重要信息
- 语义色：success/warning/error/info
- 配色比例：60-30-10原则

### 3. 字体
- 标题：36px / 30px / 24px / 20px / 18px
- 正文：16px（移动端14px）
- 行高：1.5 - 1.8
- 字重：400/500/600/700

### 4. 组件
- 按钮：高度40px/48px，圆角4px/8px
- 输入框：高度40px/48px，边框1px
- 卡片：圆角8px/12px，阴影0 2px 8px rgba(0,0,0,0.1)
- 间距：组件内16px，组件间24px

## 检查清单

- [ ] 移动端适配
- [ ] 颜色对比度符合WCAG
- [ ] 交互状态清晰（hover/active/disabled）
- [ ] 加载状态处理
- [ ] 空状态处理
- [ ] 错误提示友好
