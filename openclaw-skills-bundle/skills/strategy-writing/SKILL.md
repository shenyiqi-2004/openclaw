# 写作策略

## 写作流程

1. **思考** → 2. **大纲** → 3. **初稿** → 4. **反思** → 5. **修改**

## 反思清单

写完后自问：
- [ ] 核心观点是否清晰？
- [ ] 论据是否充分？
- [ ] 读者能理解吗？
- [ ] 有没有废话？
- [ ] 开头够吸引吗？
- [ ] 结尾有行动号召？

## 大纲模板

```markdown
# 标题

## 一、问题/背景
- 痛点是什么
- 为什么重要

## 二、核心观点
- 主要论点1
- 主要论点2
- 主要论点3

## 三、解决方案/论证
- 论点1 + 例子
- 论点2 + 例子
- 论点3 + 例子

## 四、总结
- 重申观点
- 行动号召
```

## 检查清单

- [ ] 标题吸睛
- [ ] 开头3秒抓住注意力
- [ ] 每段一个观点
- [ ] 使用具体例子
- [ ] 结论有行动号召
