# Parsing Reference

This local install is the lightweight version of the official Scrapling skill.

For the full parsing reference, see:
- https://github.com/D4Vinci/Scrapling/tree/main/agent-skill/Scrapling-Skill/references/parsing
- https://github.com/D4Vinci/Scrapling/tree/main/docs

## Quick reminder

- CSS selectors: `page.css(...)`
- XPath selectors: `page.xpath(...)`
- BeautifulSoup-like search: `find`, `find_all`, `find_by_text`
- Raw parser-only mode: `from scrapling.parser import Selector`
