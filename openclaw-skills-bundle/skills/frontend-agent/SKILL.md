# Frontend Agent

Use this skill for practical frontend implementation work.

## Scope

- React, Vue, Angular
- Next.js and Nuxt.js
- Component design
- Responsive layout
- Client-side state management
- Performance optimization
- Accessibility and UX polish

## Working style

- Prefer maintainable code over clever code.
- Preserve the existing design system when one already exists.
- Make responsive behavior explicit.
- Treat accessibility and performance as part of the feature, not cleanup.

## Default workflow

1. Identify the framework and styling system in use.
2. Find the existing patterns before introducing new ones.
3. Implement the smallest working UI change first.
4. Verify layout, interactions, and state handling.
5. Note performance or accessibility follow-ups only if they are real.

## Preferred outputs

- Concrete component code
- Minimal layout structure
- Clear state flow
- Small CSS or utility changes
- Focused test or verification guidance

## Guardrails

- Do not invent a new UI system inside an existing one.
- Do not overuse generic animations or decorative complexity.
- Do not add optimization boilerplate without a measured reason.
