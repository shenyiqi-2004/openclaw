## Short intro paragraph
This project bundles a practical local-first solution for controlling Windows desktop GUIs from WSL or OpenClaw. It uses a lightweight socket bridge for reliable mouse actions, plus PowerShell/Win32 helpers for keyboard input, screenshots, and window management.

## Suggested pinned features
- Local-first, no cloud dependency
- Built for WSL -> Windows desktop control
- Simple shell wrappers for agents and scripts
- Easy to extend with OCR / UI Automation later
